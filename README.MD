# Studio Schedule

Веб-приложение для создания и редактирования расписания занятий с экспортом в изображение.

## Для чего проект

Проект нужен для:
- ведения недельного расписания (по дням и времени);
- быстрого редактирования карточек занятий (название, тренер, зал и т.д.);
- экспорта расписания в форматах `PNG`, `JPEG`, `SVG`;
- генерации постера на день (вкладка экспорта `День`) с фоновым изображением.

## Что умеет

- режимы отображения: `Таймлайн`, `Список`, `Компактный`;
- настройка оформления: шрифты, цвета, тема, прозрачности;
- импорт/экспорт данных в `JSON`;
- автосохранение состояния в `localStorage`;
- экспорт недели и дня с предпросмотром.

## Быстрый запуск

Приложение статическое, сборка не требуется.

### Вариант 1: Python

```bash
cd webSite
python -m http.server 8080
```

Откройте в браузере:  
`http://localhost:8080`

### Вариант 2: Node.js (`serve`)

```bash
cd webSite
npx serve .
```

Откройте адрес, который покажет команда (обычно `http://localhost:3000`).

## Использование

1. Откройте `Настройки` и задайте параметры сетки/отображения.
2. Добавьте направления, тренеров и занятия.
3. При необходимости импортируйте существующий `JSON`.
4. Нажмите `Получить изображение`:
   - вкладка `Неделя` — экспорт всей таблицы;
   - вкладка `День` — экспорт постера выбранного дня.
5. Выберите формат (`PNG` по умолчанию) и скачайте файл.

## Работа с данными

- `Экспорт JSON` — сохраняет текущее состояние приложения;
- `Импорт JSON` — загружает ранее сохраненное состояние;
- состояние также автоматически хранится в `localStorage`.

## Обновление списка шрифтов

Если добавляете/меняете шрифты в папке `fonts`, обновите файл `fonts.generated.js`:

```bash
node sync-fonts.mjs
```

## Структура проекта

- `index.html` — интерфейс приложения;
- `styles.css` — стили;
- `app.js` — основная логика;
- `fonts.generated.js` — справочник доступных шрифтов;
- `sync-fonts.mjs` — генерация/синхронизация списка шрифтов.
